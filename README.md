# MovieChoice 3000

##### A comprehensive movie choosing tool for the Interested but Indecisive

### DATA MODELS:
#### User:
`username: string`
`password: string`
`id: string`
`movies: object[]`

#### Movie:
`title: string`
`description: string`
`rating: string`
`score: string`
`length: string (time?)`
`review: string`
`released: string (date)`
`platforms: string[]`
`genres: string[]`
`tags: string[]`
`posterImg: string`
`trailer: string`
`cast: string[]`
`desire to see: number (out of 10)`
`mentions: number`
